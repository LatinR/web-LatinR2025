Inteligencia Artificial Generativa en forma local. LLMs en tu PC usando ollama y ollamar

Use LLMs locais com Ollama e ollamar: instalação, uso em R e análise de textos offline

Sobre o curso

Data 1 de Dezembro de 2025, 10:00 AM - 12:00 PM (UTC-3)

Modalidade: Online

Idioma: Espanhol

Inscrição: Inteligencia Artificial Generativa en forma local

Este workshop introdutório propõe uma experiência prática com Modelos Grandes de Linguagem (LLMs) locais utilizando duas ferramentas básicas: 1) ollama, uma plataforma leve que permite descarregar e executar modelos de linguagem directamente num computador pessoal, sem necessidade de ligação à Internet nem acesso a APIs externas; 2) ollamar, uma biblioteca para integrar o R com o Ollama e facilita o trabalho com estruturas de dados (por exemplo, históricos de conversação/chat) que são padrão para diferentes LLMs.

Ao longo da sessão, os participantes serão guiados através da instalação do Ollama, carregamento de um modelo compatível (como LLaMA, Mistral ou outros disponíveis), e execução de prompts estruturados para realizar tarefas básicas de processamento de linguagem natural (NLP). Será trabalhado com um conjunto de dados de notícias em espanhol, aplicando os modelos locais para identificar o tom de uma notícia, gerar resumos, etc.

O objetivo é fornecer uma primeira abordagem técnica ao uso de LLMs sem ligação, explorando tanto o seu potencial como as suas limitações em comparação com modelos na nuvem.

Requisitos

Para participar precisa de um computador e ligação à internet. Para evitar problemas de compatibilidade e de hardware, o workshop será desenvolvido num Google Colab (com kernel R).

Instrutor

German Rosatti - Germán Rosati é sociólogo, Investigador Associado do CONICET e professor na Escola Interdisciplinar de Estudos Sociais Avançados da UNSAM, onde coordena factor~data, laboratório em Ciências Sociais Computacionais. Participou em projetos que utilizam LLMs para extração de informação em textos administrativos e jornalísticos, classificação automatizada e análise de preconceitos.

Inscrição: Forecasting with regression models

Este workshop foca nos fundamentos da previsão baseada em características com regressão linear usando R. O workshop é amigável para iniciantes e cobre os seguintes tópicos: - Como enquadrar uma série temporal como um problema de regressão - Técnicas de engenharia de características para dados com consciência temporal - Modelagem de tendência, sazonalidade, outliers e quebras - Dicas práticas para avaliação e validação

Inscrição: Intro a Positron

Venha conhecer o Positron, o novo IDE baseado no VS Code para ciência de dados com R e Python. Simples, integrado e sem configuração extra.

Este workshop introduz a comunidade do R ao IDE Positron, um ambiente baseado no Visual Studio Code projetado para ciência de dados. Ao contrário do RStudio ou VS Code com extensões, o Positron integra nativamente ferramentas que simplificam a exploração de dados e a gestão de variáveis, reduzindo a necessidade de configurações complexas.

O tutorial destina-se a cientistas de dados, analistas, estatísticos e profissionais que usam R ou Python para análise de dados e desejam otimizar seu fluxo de trabalho através de um IDE moderno e eficiente. Também é útil para professores ou estudantes avançados que procuram explorar ferramentas que facilitem a exploração de dados e a gestão de variáveis sem depender de configurações complexas.

Requisitos: Recomenda-se que os participantes tenham conhecimentos básicos de R ou Python e estejam familiarizados com a execução de scripts e projetos de análise de dados, para poder aproveitar plenamente as funcionalidades do Positron, como o Data Explorer, o Variables Pane e a integração com LLMs.

Este tutorial aproveita os créditos gratuitos da API Gemini do Google para ensinar como usar LLMs no R, um ambiente amigável para PLN.   3. Manejar dados tabulares com as principais funções do pacote dplyr (como mutate, filter, select).

Foi parte da equipe que projetou o lançamento da plataforma digital do canal, transformando-se na primeira mulher chefe da história do canal que já tem 65 anos.

É graduada do Colégio Universitário de Jornalismo Obispo Trejo y Sanabria. Foi docente da instituição nas carreiras de Jornalismo e na Licenciatura em Comunicação Social. Atualmente, é apresentadora de um noticiário e um programa de interesse geral que é transmitido de segunda a sexta-feira no Canal 12. Além disso, oferece cursos e capacitações para empresas, organizações e palestrantes que precisam desenvolver uma comunicação eficaz para diferentes contextos.
